The difference between Margin, Border, and Padding

Sprint Two Technical Blog | Sunday 22nd November, 2020

The best way to describe the difference between these three properties is to use the box model. In CSS the box model consists of layers that wrap around every HTML element. These layers are: border, padding, margin, and the element itself.

Margin

Specifies the amount of space between the border and the outside edge of the element. Each side of the margin layer can also be specified individually.

Border

Specifies the amount of space between the padding of the content area and the margin. This property specifies the width of the border along with the border style and colour.

Padding

Specifies the amount of space between the content area and the border. Each side of the padding layer can also be specified individually.

Content

The area of the HTML element itself. The size of the content is specified by width and height.
